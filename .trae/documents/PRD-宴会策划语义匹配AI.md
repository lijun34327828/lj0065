## 1. 产品概述

宴会策划语义匹配 AI 是一款智能宴会方案生成工具，用户通过选择宴会规模、举办场地、宾客人群、预算区间等基础参数，由 AI 自动匹配生成多套完整的宴会方案，包含宴会风格、配套活动、菜品档次、伴手礼类型等内容。用户可在侧边面板自由替换场地布置、娱乐项目，每次修改都会实时更新整体预算总额。

- **核心价值**：将复杂的宴会策划简化为智能匹配与可视化编辑，大幅提升策划效率
- **目标用户**：宴会策划师、企业行政人员、新人筹备团队
- **产品定位**：纯文本交互的智能宴会方案搭配工具

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 使用所有方案生成与编辑功能 |

### 2.2 功能模块

1. **参数选择区**：宴会规模、举办场地、宾客人群、预算区间四大维度选择
2. **AI 智能匹配**：根据参数自动生成多套风格迥异的完整宴会方案
3. **方案展示面板**：侧边面板展示方案明细，支持多方案切换
4. **方案编辑功能**：自由替换场地布置、娱乐项目、菜品档次等
5. **实时预算计算**：每次修改即时更新总预算与分项预算

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 顶部品牌区 | 展示产品名称、算力支持标识、3825 可视化编辑标识 |
| 主页面 | 参数选择区 | 四个维度的参数卡片式选择，支持单选与交互反馈 |
| 主页面 | AI 生成按钮 | 触发方案生成，带有加载动效 |
| 主页面 | 方案侧栏 | 多套方案列表，可切换查看 |
| 主页面 | 方案详情区 | 展示当前选中方案的完整明细 |
| 主页面 | 预算面板 | 实时显示总预算与各分项预算 |

## 3. 核心流程

用户打开页面 → 选择宴会规模 → 选择举办场地 → 选择宾客人群 → 选择预算区间 → 点击 AI 匹配 → 系统生成 3 套方案 → 用户选择方案查看详情 → 用户替换场地布置/娱乐项目 → 实时更新预算 → 完成方案搭配

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深酒红 `#722F37` 搭配鎏金 `#D4AF37`，营造奢华宴会氛围
- **辅助色**：象牙白 `#FFFFF0` 背景，墨黑 `#1A1A1A` 文字
- **按钮风格**：精致圆角，金边描边，悬停有微光泽动效
- **字体搭配**：标题使用 Playfair Display（衬线奢华感），正文使用 Lato（现代易读）
- **布局风格**：左右分栏主布局 + 卡片式参数选择 + 侧边方案列表面板
- **视觉层次**：金边装饰线条、微妙纹理背景、柔和阴影营造高级感

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主页面 | 顶部品牌区 | 品牌 Logo、产品名称、算力标识徽章 |
| 主页面 | 参数选择区 | 四个维度卡片组，选中态有金边高亮 |
| 主页面 | AI 生成按钮 | 大尺寸渐变按钮，加载时有流光动效 |
| 主页面 | 方案侧栏 | 竖向方案卡片列表，选中项突出显示 |
| 主页面 | 方案详情区 | 分类折叠面板，每项可单独编辑替换 |
| 主页面 | 预算面板 | 底部悬浮固定，大字号显示总金额 |

### 4.3 响应式

- 桌面端优先设计，适配 1440px 及以上宽度
- 平板端：参数区改为 2x2 网格布局
- 移动端：方案侧栏改为底部抽屉式弹出
- 触控优化：按钮最小尺寸 44px，间距适当放大

### 4.4 动效设计

- 页面加载：元素渐入 + 轻微上移，错峰出现
- 方案生成：骨架屏脉冲动画 + 进度条
- 预算更新：数字滚动过渡效果
- 卡片选中：缩放 + 边框发光过渡
- 悬停反馈：微妙的阴影加深与背景色变化
